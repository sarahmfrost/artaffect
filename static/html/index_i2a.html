This is the Image to Affect project, developed by Ruben Calderon, Ty Burnham, and Sarah Frost. This project relies on Machine Learning to develop a model will take an image and return the affect vector of that image. The model has been trained on our initial database of 4000 images of fine art. The “Sorting Game” feature allows the user to get an intuitive understanding of the underlying AI system. The “User Upload” functionality gives immediate feedback on the emotional content of a piece of art chosen by the user. The code for this section can be seen here.
